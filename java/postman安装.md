# 安装步骤

>* 下载类似Postman_v4.9.2.crx以crx为后缀的文件
>* 改成“zip”或"rar"文件，解压该文件
>* 将文件夹里的“_metadata”文件夹名字改成“metadata”
>* 打开chrome浏览器，单击右上角“≡”图标--> “更多工具” --> “扩展程序”
>* 加载“已解压的扩展程序”，选中解压的文件夹
>* 点击加载后的postman界面上的“启动”即可。
